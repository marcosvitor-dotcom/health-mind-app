# Health Mind App

Aplicativo mobile para suporte e acompanhamento em saúde mental, desenvolvido com React Native e Expo.

Um produto **Losning Tech** (CNPJ: 61.661.169/0001-87).

## Sobre o Produto

O Health Mind App é uma plataforma completa que conecta psicólogos, clínicas e pacientes, oferecendo ferramentas para gestão de saúde mental de forma integrada e segura.

## Funcionalidades

### Para Pacientes
- **Diário Terapêutico (Chat com IA)**: Assistente terapêutica digital baseada em inteligência artificial, configurada pelo psicólogo responsável, que atua como suporte complementar ao tratamento
- **Agendamento de Consultas**: Visualização, agendamento e gerenciamento de consultas com o psicólogo
- **Meu Psicólogo**: Acesso ao perfil e informações do profissional responsável pelo acompanhamento
- **Minha Clínica**: Visualização dos dados da clínica vinculada e seus profissionais
- **Emergência**: Acesso rápido a recursos de ajuda imediata (CVV: 188, SAMU: 192)
- **Perfil e Configurações**: Gerenciamento de dados pessoais, contato de emergência e preferências do app
- **Notificações Push**: Lembretes de consultas e comunicações importantes

### Para Psicólogos
- **Gestão de Pacientes**: Lista completa de pacientes com detalhes de acompanhamento
- **Agenda**: Gerenciamento de horários disponíveis e consultas agendadas
- **Documentos**: Criação e gerenciamento de anamneses, relatórios de sessão e avaliações psicológicas
- **Relatórios de IA**: Análise dos chats dos pacientes com a assistente terapêutica digital
- **Configuração da Assistente IA**: Personalização do comportamento da assistente para cada paciente
- **Perfil Profissional**: Gerenciamento de especialidades, CRP e dados profissionais

### Para Clínicas
- **Visão Geral**: Dashboard com estatísticas gerais da clínica (pacientes, consultas, profissionais)
- **Gestão de Psicólogos**: Cadastro, vinculação e gerenciamento dos profissionais da clínica
- **Agenda Geral**: Visualização consolidada da agenda de todos os profissionais
- **Gestão Financeira**: Acompanhamento de faturamento e pagamentos
- **Dados da Clínica**: Gerenciamento de informações cadastrais e endereço

## Tecnologias

- **React Native** com **Expo** (SDK 54) para desenvolvimento multiplataforma
- **TypeScript** para tipagem estática
- **Node.js** com **Express** para o backend
- **MongoDB** para banco de dados
- **JWT** para autenticação segura com refresh tokens
- **Expo Push Notifications** para notificações em tempo real
- **Inteligência Artificial** para a assistente terapêutica digital

## Tipos de Acesso

O aplicativo possui 3 perfis de usuário:

1. **Paciente** - Para pessoas que recebem acompanhamento psicológico
2. **Psicólogo** - Para profissionais de saúde mental registrados no CRP
3. **Clínica** - Para administração e gestão de clínicas de psicologia

## Scripts Disponíveis

- `npm start` - Inicia o servidor de desenvolvimento
- `npm run android` - Executa no Android
- `npm run ios` - Executa no iOS
- `npm run web` - Executa no navegador
